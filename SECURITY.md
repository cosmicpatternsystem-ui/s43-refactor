# Security Policy

## Supported Versions
| Version | Supported |
| --- | --- |
| v1.0.0-GOLD | Yes |

## Reporting a Vulnerability
Any breach in the `ASO_CORE_LOGIC` or `GOVERNANCE_RISK_GUARD_CONTRACT` must be reported immediately to the core maintainers.

## Integrity Audit
The system uses automated integrity checks located in `runtime/audit/`. Any tampering with the core codebase triggers an automatic lockdown of the `PRIME-X1` engine.
